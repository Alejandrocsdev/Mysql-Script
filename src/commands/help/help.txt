===============
mysqlScript CLI
===============

./run.sh <command> [argument]

Commands:
  help                    Show help message
  help:install            Show the MySQL installation steps
  help:system             Show commonly used systemctl commands for MySQL

  user:show               List created MySQL users
  user:create <arg>       Create a MySQL user
                          <arg>: user:<user>,host:<host>,pass:<password>
  user:drop <arg>         Drop a MySQL user
                          <arg>: user:<user>,host:<host>
  user:db:show <arg>      List databases accessible by user
                          <arg>: user:<user>,host:<host>

  db:show                 List created MySQL databases
  db:create <arg>         Create a MySQL database
                          <arg>: name:<dbName>
  db:drop <arg>           Drop a MySQL database
                          <arg>: name:<dbName>
  db:grant <arg>          Grant database access to user
                          <arg>: name:<dbName>,user:<user>,host:<host>
  db:revoke <arg>         Revoke database access from user
                          <arg>: name:<dbName>,user:<user>,host:<host>
  db:user:show <arg>      List users with access to database
                          <arg>: name:<dbName>

  config                  Open MySQL server config in nano editor
